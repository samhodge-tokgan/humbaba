# Test assets

Media here is **gitignored** (large binaries) and fetched on demand. This file and
any `manifest.*` are tracked.

## Fixtures

- `synthetic_test.png` — generated by `tests/natron/make_test_image.py`; hermetic, used
  by the passthrough render test. Not committed.

## Real assets (wired in with M3)

- **Still:** Kodak "Marcie" (the Digital LAD reference image) — used to eyeball depth output.
- **Sequence:** a <24-frame clip from an **ASWF open dataset** (e.g. an ACES/OpenEXR sample
  sequence), read as ACEScg RGBA float, for the image-sequence tests in M4.

A fetch script and pinned URLs land in M3, where the model actually runs and the depth
output is inspected.
